# Project Brief

## Core Requirements
* Aggregate job postings from portals (TopCV, LinkedIn, Joboko).
* Core: career intelligence hub.
* Features:
  * Fast job search.
  * Interactive data visualizations (market trends).
  * AI chatbot assistant.
* Robust web backend.
  * Scrape pipeline.
  * Big data handling.
  * Secure storage (CVs, profile, chat logs).
  * High-concurrency ML microservice gateway.

## Constraints & Objectives
* Fast First Contentful Paint (FCP).
* High concurrency.
* Secure personal info.
